# What is MDX?

## How to use Component in MDX

```tsx
// _components/examples.tsx
import { CardBasic } from '@/app/_components/cards'

export function ExampleComponent() {
  return <CardBasic title="title" body={<p> Body </p>} footer="footer" />
}

// mdx/page.mdx
import { ExampleComponent } from './_components/examples'
;<ExampleComponent />
```

# What is this?

[Next.js](https://nextjs.org/)를 기반으로 제작되었으며 애버커스의 프론트엔드 개발자들이 보다 빠르게 리액트 프로젝트를 시작 할 수 있도록 도와주는 템플릿입니다.

---

## 시작하기

Node.js 버전 "18.12" 이상과 PNPM 버전 "8" 이상이 필요합니다.
최신 버전의 Node.js를 설치하려면 [Node.js 공식 홈페이지](https://nodejs.org/ko/)를 참고하세요.

### 1. 프로젝트 클론

```bash
git clone https://gitlab.abacussw.co.kr/aiplatform/abacus-common-framework/framework-next.git
```

### 2. 의존성 설치

```bash
pnpm install
```

### 3. 개발 서버 실행

```bash
pnpm dev
```

---

## 새로운 페이지 추가하기

### 1. 페이지 생성

`app/[lang]/{moudule-name}` 디렉토리에 page.tsx 파일을 생성합니다.
아래 명령어의 test 부분을 원하는 페이지 이름으로 변경합니다.

```bash
moduleName=test
mkdir -p app/[lang]/$moduleName
touch app/[lang]/$moduleName/page.tsx
code app/[lang]/$moduleName/page.tsx
```

---

### 2. 페이지 컴포넌트 작성

아래 코드를 붙여넣습니다.

```tsx
import { subtitle, title } from '@/app/_components/server-only/primitives'
import { AVAIL_LOCALES, TAvailLocale } from '@/config/system'
import { getDictionary } from '@/app/[lang]/dictionaries'

export async function generateStaticParams() {
  return AVAIL_LOCALES.map((lang) => ({ lang }))
}

interface Param {
  params: { lang: TAvailLocale }
}

export default async function SSGPage({ params: { lang } }: Param) {
  const dict = await getDictionary(lang)
  return (
    <section className="flex flex-col items-center justify-center gap-4 py-8">
      <div className="inline-block max-w-lg text-center justify-center">
        <h1 className={title()}>Welcome to &nbsp;</h1>
        <h1 className={title({ color: 'violet' })}>Hi?</h1>
        <br />
        <h2 className={subtitle({ class: 'mt-4' })}>by Abacus</h2>
      </div>
    </section>
  )
}
```

### 3. 컴포넌트 선정 및 삽입

이미 프레임워크는 예시 컴포넌트 목록 페이지를 제공하고 있습니다.

1. `pnpm storybook` 명령어를 통해 storybook 서버를 실행합니다.
2. http://localhost:6006/ 에 접속하여 원하는 컴포넌트를 선정, 삽입합니다.

## 프로젝트 테마 설정
우리가 마주하는 가장 큰 과제중 하나는 프로젝트의 테마를 설정하는 것입니다.  
백날 잘 만들어둔 컴포넌트들이 있지만, 테마를 설정하지 않으면 그 컴포넌트들은 아무런 의미가 없습니다.  
우리는 프로젝트의 테마를 설정하기 위해, `app/_components/server-only/primitives.ts` 파일을 제공합니다.
--- 
### 폰트 변경하기
폰트를 변경하기 위해서는 `app/_components/server-only/primitives.ts` 파일을 열어, `font` 함수를 찾습니다.
### 버튼의 radius 변경하기
버튼의 radius를 변경하기 위해서는 `app/_components/server-only/primitives.ts` 파일을 열어, `button` 함수를 찾습니다.
### 전체 테마의 주요 컬러를 변경하기
전체 테마의 주요 컬러를 변경하기 위해서는 `app/_components/server-only/primitives.ts` 파일을 열어, `color` 함수를 찾습니다.

## Linting

애버커스는 [ESLint](https://eslint.org), [Prettier](https://prettier.io/) 를 사용하여 코드를 검사, 통일화합니다.  
모든 코드를 린트하려면 다음을 실행할 수 있습니다.

```bash
pnpm format:fix
```

### ESLint

We recommend installing the [ESLint plugin for VS Code](https://marketplace.visualstudio.com/items?itemName=dbaeumer.vscode-eslint)

You can find the enabled rules in the [ESLint config](https://github.com/vercel/next.js/blob/canary/.eslintrc.json)

### Prettier

We recommend installing the [Prettier plugin for VS Code](https://marketplace.visualstudio.com/items?itemName=esbenp.prettier-vscode)

You can find the format configuration in the [Prettier config](https://github.com/vercel/next.js/blob/canary/.prettierrc.json)

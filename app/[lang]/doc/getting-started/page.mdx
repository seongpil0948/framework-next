# What is this?

[Next.js](https://nextjs.org/)를 기반으로 제작되었으며 애버커스의 프론트엔드 개발자들이 보다 빠르게 리액트 프로젝트를 시작 할 수 있도록 도와주는 템플릿입니다.

---

## 시작하기

### 1. 프로젝트 클론

```bash
git clone https://gitlab.abacussw.co.kr/aiplatform/abacus-common-framework/framework-next.git
```

### 2. 의존성 설치

```bash
pnpm install
```

### 3. 개발 서버 실행

```bash
pnpm dev
```

---

## 새로운 페이지 추가하기

### 1. 페이지 생성

`pages` 디렉토리에 새로운 페이지를 생성합니다.

### 2. 라우팅 설정

`src/routes.ts` 파일에 라우팅 설정을 추가합니다.

### 3. 페이지 컴포넌트 작성

## `src/pages` 디렉토리에 생성한 페이지 컴포넌트를 작성합니다.

---

## 새로운 컴포넌트 추가하기

### 1. 컴포넌트 생성

`src/components` 디렉토리에 새로운 컴포넌트를 생성합니다.

### 2. 컴포넌트 스타일 작성

`src/components/styles` 디렉토리에 컴포넌트 스타일을 작성합니다.

### 3. 컴포넌트 스토리북 작성

## `src/components/stories` 디렉토리에 컴포넌트 스토리북을 작성합니다.

---

## 결론

이제 애버커스의 프론트엔드 개발자들은 리액트 프로젝트를 시작할 때마다 프로젝트 구조를 만들고, 라우팅을 설정하고, 스타일을 작성하는 등의 반복적인 작업을 하지 않아도 됩니다. 이제부터는 이 템플릿을 사용하여 프로젝트를 시작하면 됩니다.
